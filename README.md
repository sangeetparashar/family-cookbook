## Setup

1. Clone the repository:

   ```bash
   git clone [your-repo-url]
   cd recipe-viewer
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Add your PDF files to public/recipes

4. Run the dev server:

   ```bash
   npm run dev
   ```
